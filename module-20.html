За пределами кода: ЦА, продукт, аналитика

Claude Code умеет не только писать код: анализ целевой аудитории, дизайн и аналитика продукта, контент.

20/20
100%
🎯 Результат модуля

Ты используешь Claude Code как продуктового партнёра, а не только как кодера.

❙ Анализ целевой аудитории

Дай Claude доступ к данным (через MCP: аналитика, заметки интервью, отзывы) — и он соберёт портреты ЦА, боли и формулировки ценности.

Проанализируй продукт по @README.md и отзывам @feedback.md.
Собери 3 портрета ЦА: кто это, их задачи (jobs-to-be-done),
боли, возражения и триггеры покупки. Для каждого — одна
фраза ценностного предложения. Запиши в audience.md.

❙ Bulletproof-промпты под продукт

Анализ конкурентов
Изучи конкурентов [список]. Сравни по: позиционирование,
функции, цена, слабые места. Где наша возможность отстроиться?
Таблица + 3 вывода.
Гипотезы роста
На основе audience.md предложи 5 гипотез роста по ICE
(Impact, Confidence, Ease). Отсортируй по приоритету и
опиши, как проверить каждую за неделю.
Аудит лендинга (CRO)
Проверь @index.html как лендинг: ясность оффера, перваый
экран, призывы к действию, возражения, доверие. Дай список
правок по приоритету влияния на конверсию.
