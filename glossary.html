Глоссарий

Глоссарий: термины простыми словами

Все «страшные» слова из курса — на человеческом языке. Загляни сюда, если что-то непонятно.

Основы

CLI / терминал
Текстовое окно, где программам дают команды словами, а не мышкой.
Команда
Строчка-приказ программе. Например, claude запускает агента.
Флаг
Настройка команды через дефис. --model opus = «работай на модели opus».
Сессия
Один разговор с Claude. Можно продолжить старую: --continue.
Headless (-p)
Запуск без диалога: дал задачу — получил ответ. Нужен для скриптов и автоматизаций.
Repo / репозиторий
Папка проекта под управлением git, где хранится код и его история.

Память и контекст

Контекст
«Рабочая память» разговора: всё, что Claude сейчас видит. Переполнишь — хуже отвечает.
Токен
Кусочек текста, в которых считается объём и цена. Меньше лишнего в контексте — дешевле.
CLAUDE.md
Файл-памятка проекту: команды, стиль, правила. Claude читает его в начале каждой сессии.
/compact / /clear
Сжать длинный разговор / стереть и начать с чистого листа.

Как управлять

Промпт
Твоя задача, написанная словами. Чем точнее — тем лучше результат.
Режим разрешений
Насколько Claude действует сам: спрашивать на всё / править самому / только план.
Plan mode
Режим «сначала план»: Claude показывает, что собирается делать, до изменений.
ultrathink
Слово-триггер «думай по максимуму» для сложных задач (есть think → think hard → ultrathink).
Ревью (review)
Проверка кода свежим взглядом — на баги, безопасность, читаемость.

Расширения

Skill (навык)
Сохранённая инструкция «как делать задачу». Claude подключает её сам, когда нужно.
Субагент
Отдельный помощник со своей памятью. Делает рутину и приносит результат.
Плагин
Набор расширений «в один клик»: команды + агенты + инструменты сразу.
Маркетплейс
«Магазин» плагинов — обычно git-репозиторий, откуда их ставят.
MCP
«Переходник» к внешним инструментам: браузер, GitHub, база данных, Figma.

Автоматизация

Хук (hook)
Правило «если событие — то действие». Например: сохранил файл → запусти тесты.
Cron / расписание
Запуск задачи по времени: «каждое утро в 9:00». Claude делает сам.
Loop
Повтор задачи по кругу до условия. Например, чинить тесты, пока не зелёные.
Worktree
Отдельная копия-ветка проекта. Удобно пробовать рискованное, не ломая основное.
Hivemind / рой
Много агентов параллельно над частями одной большой задачи.

Сохранность

Git / коммит
Система версий. Коммит — «точка сохранения», к которой всегда можно вернуться.
/rewind
Откатить проект к прошлому состоянию прямо внутри сессии.
CI
Автозапуск проверок на сервере при изменениях кода (например, в GitHub).

Бизнес-термины (в промптах)

ICP
Портрет идеального клиента: кто это, его боли и задачи.
JTBD
«Работа», ради которой клиент нанимает продукт: «когда…, хочу…, чтобы…».
УТП
Уникальное торговое предложение — почему выбирают тебя, а не других.
CRO
Оптимизация конверсии — как превратить больше посетителей в клиентов.
