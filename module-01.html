Модуль 01

Новая парадигма: агент в терминале

Понять, что такое Claude Code и почему это не «чат с кодом», а агент, который сам читает проект, пишет файлы и запускает команды.

01/20
5%
🎯 Результат модуля

Ты понимаешь, чем Claude Code отличается от Cursor и ChatGPT, и когда он сильнее всего.

Простыми словами

Claude Code — это очень умный стажёр прямо в твоём компьютере: ты объясняешь словами, что нужно, а он сам открывает файлы, пишет код и запускает команды.

❙ Простыми словами

Claude Code — это AI-агент, который живёт прямо в твоём терминале. Ты пишешь задачу обычным языком, а он сам: читает нужные файлы, редактирует код, запускает тесты, делает коммиты и объясняет, что сделал.

Главное отличие от обычного чата: Claude Code видит весь проект и действует, а не просто советует. Это «руки», а не только «голова».

✗ Чат (ChatGPT)

Ты копируешь код туда-сюда, объясняешь контекст вручную, теряешь связь с проектом.

✓ Агент (Claude Code)

Ты ставишь задачу — он сам находит файлы, меняет их, проверяет результат и коммитит.

❙ Где Claude Code сильнее всего

Большие задачи

Рефакторинг, новая фича через много файлов, миграции.

multi-file

Автономность

Может работать долго сам: план → код → тесты → фикс.

agentic

Интеграции

Подключается к GitHub, браузеру, Figma, БД через MCP.

MCP

Автоматизация

Хуки, расписания, CI — работа без твоего участия.

hooks

❙ Три кита, на которых всё держится

Контекст — Claude должен видеть проект и правила (об этом модули 06–07).
Разрешения — ты контролируешь, что он может делать сам (модуль 04).
Проверка — даёшь способ проверить результат: тесты, сборку, скриншот.
🔑
Золотое правило курса: всегда давай Claude способ проверить себя. Тогда он работает сам, а ты не превращаешься в ручной «детектор ошибок».
