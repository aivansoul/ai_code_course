Простое объяснение

Зачем GitHub

Чтобы не потерять работу, видеть историю, откатываться назад и публиковать проект.

Минимум команд

init, add, commit, push. Для старта этого достаточно.

Раскрывающиеся примеры и copy-paste

Команды
git init
git add .
git commit -m "Initial version"
git branch -M main
git remote add origin https://github.com/username/project-name.git
git push -u origin main
Скопируйте блок целиком и вставьте в Cursor, ChatGPT, Claude или терминал, если это команды.

Практика шаг за шагом

Создать репозиторий.
Сделать первый commit.
После каждого большого изменения делать новый commit.
