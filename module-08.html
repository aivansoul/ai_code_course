Модуль 08

Промпты для Claude Code

Освоить рабочую формулу задач и легендарный цикл «Explore → Plan → Code → Commit».

40%
🎯 Результат модуля

Ты формулируешь задачи так, что Claude с первого раза делает то, что нужно.

Простыми словами

Промпт — это просто твоя задача словами. Хороший промпт = цель + детали + как проверить. Плохой = «сделай хорошо».

❙ Формула задачи

✗ Плохой промпт

«Сделай красиво» — даёт случайный результат.

✓ Хороший промпт

Цель + контекст + ограничения + способ проверки.

Цель: добавить тёмную тему на сайт.
Контекст: React + Tailwind, файлы в src/.
Ограничения: без новых библиотек, доступность AA.
Проверка: запусти тесты и сделай скриншот до/после.

❙ Главный цикл: Explore → Plan → Code → Commit

Explore — «изучи, как сейчас устроена авторизация, ничего не меняй».
Plan — «составь план изменений» (включи plan mode, модуль 09).
Code — «реализуй план по шагам, после каждого запускай тесты».
Commit — «закоммить логичными кусками с понятными сообщениями».
💡
Не давай кодить сразу. Сначала «изучи» и «спланируй» — это убирает 80% переделок.

❙ Режимы мышления

Для сложных задач включай расширенное мышление словами-триггерами. Сила растёт по нарастающей:

thinkthink hardthink harderultrathink — максимум рассуждений
Спроектируй схему БД для мультитенант-SaaS. ultrathink: продумай изоляцию данных, индексы и миграции.
