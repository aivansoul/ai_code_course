Раскрывающиеся примеры и copy-paste

Минимальный HTML шаблон
<!doctype html>
<html lang="ru">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>AI Code Start</title>
  <link rel="stylesheet" href="style.css" />
</head>
<body>
  <main>
    <section class="hero">
      <h1>AI Code Start</h1>
      <p>Экспресс-курс по работе с AI-инструментами для разработки.</p>
      <a href="#program" class="button">Смотреть программу</a>
    </section>
  </main>
  <script src="script.js"></script>
</body>
</html>
Скопируйте блок целиком и вставьте в Cursor, ChatGPT, Claude или терминал, если это команды.
