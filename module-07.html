Простое объяснение

Что такое Rules

Rules — это постоянные инструкции для AI внутри проекта. Они помогают не повторять каждый раз одни и те же требования.

Где хранить

В Cursor можно создать папку .cursor и файл rules или использовать механизм Project Rules в интерфейсе, в зависимости от версии.

Раскрывающиеся примеры и copy-paste

Пример rules
# Cursor Rules

- Пиши простой, читаемый код.
- Используй чистый HTML, CSS и JavaScript, если не сказано иначе.
- Не используй Bootstrap.
- Не используй эмодзи.
- Дизайн: минимализм, белый фон, electric blue, много воздуха.
- Все блоки должны быть адаптивными.
- Каждый сложный фрагмент сопровождай коротким комментарием.
- Не удаляй существующий функционал без причины.
Скопируйте блок целиком и вставьте в Cursor, ChatGPT, Claude или терминал, если это команды.

Практика шаг за шагом

Создать .cursor/rules.md.
Добавить правила.
Попросить Cursor проверить проект на соответствие rules.
