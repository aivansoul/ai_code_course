Как пользоваться Cursor: Ask, Edit, Agent

Понять 3 режима работы и не путать их.

Результат модуля

После этого блока человек понимает, что делать руками, что поручать AI и какой следующий шаг сделать в проекте.

Простое объяснение

Ask

Используется, когда нужно объяснение. Например: “Объясни, почему у меня не работает адаптивность”.

Edit

Используется, когда нужно изменить конкретный файл или выбранный фрагмент.

Agent

Используется, когда нужно выполнить задачу по проекту: создать структуру, добавить страницы, исправить несколько файлов.
