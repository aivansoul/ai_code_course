Plugin: действие во внешнем сервисе

Понять, что plugin делает, а skill направляет мышление.

Результат модуля

После этого блока человек понимает, что делать руками, что поручать AI и какой следующий шаг сделать в проекте.

Простое объяснение

Разница

Skill говорит AI, как думать. Plugin даёт AI возможность что-то сделать: создать дизайн, отправить письмо, прочитать календарь, открыть GitHub.

Пример

Canva plugin помогает создать визуал. GitHub plugin помогает работать с репозиторием. Gmail plugin помогает с письмами.

Раскрывающиеся примеры и copy-paste

Короткая формула
Skill = метод мышления.
Plugin = инструмент действия.
MCP = способ подключения инструментов.
Скопируйте блок целиком и вставьте в Cursor, ChatGPT, Claude или терминал, если это команды.

Практика шаг за шагом

Запомнить разницу.
Для старта выбрать не больше 2–3 plugin.
Не перегружать рабочее пространство.
