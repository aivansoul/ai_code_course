CODE for women — Claude Code Mastery (pixel-art мини-курс)

Открой index.html в браузере. Все 20 модулей + Бонусы + Шпаргалка работают локально.

Контент централизован:
  content.mjs  — тексты всех модулей, бонусов, шпаргалки
  sprites.mjs  — пиксель-арт иконки и hero-сцена (SVG)
  build.mjs    — генератор статических страниц
  styles.css   — пиксельная дизайн-система
  app.js        — кнопки COPY и анимация прогресс-баров

Пересобрать сайт после правок контента:
  node build.mjs

Публикация: GitHub Pages (уже настроен) — просто git push.
